# Alfred-workflow-default-volume
An Alfred workflow to set the output volume on your Mac to your desired default

Type `defvol` (or other keyword if you have changed the default) and ‚èé to reset the sound output of your Mac to the default. You can change the default sound level in the User Configuration to anything between 0 (which means muted) to 100 (the meximum).
